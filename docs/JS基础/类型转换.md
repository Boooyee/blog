### 原始值转数字

用Number函数将类型转为数字类型，如果无法转为数字，返回NaN。

undefined > NaN

null > 0

Boolean > 参数为true返回1，false返回0

number > 返回对应的值

string > 先转换成整数或者浮点数，忽略前置0，如果一个字符不是数字，返回NaN。

### 原始值转字符

undefined > "undefined"

null > "null"

boolean > 'true' || 'false'

string > 不变

number > 0 -0 转换成'0',NaN 转换成‘NaN’

调用String()/Number()/Boolean()等构造函数，转换成对应的包装类型。
