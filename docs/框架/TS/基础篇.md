### 动态类型与静态类型

* 强类型语言：不允许改变变量类型
* 弱类型语言：变量允许赋值为不同类型

* 静态类型语言：编译阶段确定变量类型
  + 特点：
    - 类型极度严格
    - 立即发现错误
    - 运行时性能好
    - 自文档化
* 动态类型语言：执行阶段确定所有变量类型
  + 在程序运行时，动态计算属性偏移量
  + 需要额外的空间存储属性名
  + 所有对象的偏移量信息各存一份

### 安装

+ 全局安装 `npm i typescript -g`
+ 安装成功后可以使用`tsc`  
+ 新建文件夹`TS` `npm init` `tsc init`初始化

+ 安装webpack， `npm i webpack webpack-cli webpack-dev-server -D`
